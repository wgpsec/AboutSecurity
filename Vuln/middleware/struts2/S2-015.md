## 漏洞描述

漏洞产生于配置了 Action 通配符 *，并将其作为动态值时，解析时会将其内容执行 OGNL 表达式。当配置 `<action name="*" class="..."><result>/{1}.jsp</result></action>` 时，提取 name 并解析时会对 name 执行 OGNL 表达式解析，导致命令执行。

## 影响版本

- Apache Struts 2.0.0 - 2.3.14.2

## 前置条件

- 无需认证
- 目标应用使用了 Action 通配符配置

## 利用步骤

1. 发送带有 OGNL 表达式的 action 名称
2. 触发 OGNL 表达式解析

2. 避免使用 Action 通配符 * 配置
